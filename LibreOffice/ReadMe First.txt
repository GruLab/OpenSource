1. Install latest JDK
2. Install Python (3.4.3 / 3.4.3.amd64)
3. Install pywin32 (219.win32-py3.4 / 219.amd64-py3.4)
4. Run Selenium.bat & ddt.bat as Administrator.
5. Change Data.csv file according to your credentials & language assigned.
